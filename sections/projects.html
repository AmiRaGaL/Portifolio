SDoH Prediction using LLMs (Thesis Project)

Built a robust pipeline for predicting 15 Social Determinants of Health from long clinical texts using transformer-based architectures such as Longformer, Deepseek-R1, and LLaMA 3.2.

  • Designed multi-output regression and ordinal classification models for granular SDoH prediction.
  • Implemented a hybrid Huber-MSE loss function and fine-tuned models using instruction-tuned datasets.
  • Applied active learning and uncertainty sampling strategies to improve annotation efficiency and model performance.
  • Achieved 91.3% validation accuracy and 0.285 MAE; predictions validated against clinician annotations.
